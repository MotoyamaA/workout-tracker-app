# 筋トレ記録アプリ v2.0 💪

TypeScript対応の本格的な筋トレ記録アプリです。日々のトレーニングを記録し、カロリー計算、自動提案機能、プロフィール管理、基礎代謝計算などの高度な機能を備えています。

## 🚀 v2.0 の新機能

### ✨ 主要な新機能
- **TypeScript対応** - 型安全性と開発効率の向上
- **プロフィール管理** - 身長・体重・年齢から基礎代謝を自動計算
- **カスタムエクササイズ** - 独自のエクササイズを追加・編集
- **ダークモード** - 目に優しいダークテーマ対応

- **データインポート・エクスポート** - バックアップ・復元機能
- **詳細統計** - 週間アクティビティグラフと部位別分析
- **設定画面** - テーマ、言語、単位などのカスタマイズ

### 🔧 技術的改善

## 🎯 機能一覧

### ✅ 基本機能
- **日々の記録**
  - 日付選択でワークアウトを記録
  - セット数、回数、重量、休憩時間を詳細入力
  - メモ機能でトレーニングの調子を記録

- **部位別トレーニングメニュー**
  - 胸、背中、脚、腕、肩、腹筋、有酸素の7つの部位
  - 各部位に豊富なエクササイズライブラリ
  - カスタムエクササイズの追加・編集・削除

- **カロリー計算**
  - エクササイズごとの基礎カロリー消費量
  - 重量・セット数・回数による動的計算
  - リアルタイムでの総消費カロリー表示

- **自動提案機能**
  - 過去30日の記録を分析
  - 最も頻度の低い部位を推奨
  - バランスの取れたトレーニング計画をサポート

### 🔥 高度な機能
- **プロフィール管理**
  - 個人情報（名前、年齢、性別、身長、体重、活動レベル）
  - Harris-Benedict式による基礎代謝自動計算
  - BMIと健康カテゴリーの表示
  - 総消費エネルギー量（TDEE）の算出

- **詳細統計・分析**
  - 週間アクティビティカレンダー
  - 部位別カロリー消費分析
  - 長期トレンドの可視化
  - パフォーマンス指標の追跡

- **データ管理**
  - 完全なデータインポート・エクスポート
  - JSONフォーマットでのバックアップ
  - 複数デバイス間でのデータ同期対応
  - データ削除・復元機能

- **カスタマイズ**
  - ライト・ダークテーマ切り替え
  - 単位設定（kg/lbs、cm/ft）
  - 通知設定
  - プライバシー設定

## 🚀 セットアップ

### 前提条件
- Node.js v18以上
- npm または yarn

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/MotoyamaA/workout-tracker-app.git
cd workout-tracker-app

# feature/typescript-enhancedブランチに切り替え
git checkout feature/typescript-enhanced

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

### ビルド

```bash
# TypeScriptタイプチェック
npm run type-check

# プロダクション用ビルド
npm run build

# ビルド結果をプレビュー
npm run preview
```

## 📝 使い方

### 初回セットアップ
1. **プロフィール設定** - 基本情報を入力して基礎代謝を計算
2. **エクササイズ確認** - デフォルトエクササイズを確認・カスタマイズ
3. **設定調整** - テーマや単位を好みに設定

### 日常の使用
1. **ワークアウト画面**でエクササイズを選択
2. セット数、回数、重量、休憩時間を入力
3. カロリーが自動計算される
4. メモを追加して「保存」
5. **履歴画面**で過去の記録を分析
6. **ホーム画面**で統計と推奨部位を確認

### データ管理
- **エクスポート**: 設定画面からJSONファイルでバックアップ
- **インポート**: 他のデバイスからデータを復元
- **削除**: 必要に応じて全データをリセット

## 📊 健康指標の計算

### 基礎代謝量（BMR）
```typescript
// Harris-Benedict式（改良版）
男性: BMR = 88.362 + (13.397 × 体重kg) + (4.799 × 身長cm) - (5.677 × 年齢)
女性: BMR = 447.593 + (9.247 × 体重kg) + (3.098 × 身長cm) - (4.330 × 年齢)
```

### 総消費エネルギー量（TDEE）
- 座りがち: BMR × 1.2
- 軽い活動: BMR × 1.375
- 適度な活動: BMR × 1.55
- 活発: BMR × 1.725
- 非常に活発: BMR × 1.9

### BMI計算
```typescript
BMI = 体重(kg) ÷ (身長(m) × 身長(m))
```
